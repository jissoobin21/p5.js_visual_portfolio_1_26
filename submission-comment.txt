# 제출 코멘트

- 사용한 핵심 AI / 프롬프트(선택)
  - ChatGPT를 이용해 과제 페이지 구조 정리(2×2 배치), 반응형 스타일, 가독성/심미성 개선, 접근 안내 문구를 보완했습니다.

- 사용한 핵심 HTML / CSS / Javascript
  - HTML: 각 과제 카드를 a 링크로 감싸는 섹션 구성, 과제별 iframe 임베드 페이지 생성
  - CSS: 2×2 그리드 레이아웃, 비율 유지(aspect-ratio), 카드 스타일링, 화면 폭 축소 시 1열 반응형 전환, 배경 그라디언트
  - JavaScript: 각 과제의 `sketch.js`를 로컬로 반영하여 p5.js 캔버스를 직접 렌더링, 카드 내 즉시 미리보기 구현
