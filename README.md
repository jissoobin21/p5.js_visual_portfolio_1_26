# p5.js 작품 포트폴리오 (과제 5)

## 구성

- `/index.html` : 과제 포트폴리오 메인(2×2 그리드)
- `sample1 ~ sample4` : 과제 1~4 임베드 페이지
- 각 과제 페이지는 원본 p5.js 스케치 코드를 로컬 `sketch.js`로 로드해 즉시 렌더링

## 제출 형식 요건 반영

- 과제 배치 순서:  
  - 1행: 과제 1 / 과제 4  
  - 2행: 과제 2 / 과제 3
- 링크만 텍스트로 배치하지 않고, 페이지 접속 즉시 작품이 보이도록 구성
- 과제 페이지에 원본 링크 보조 버튼을 유지해 접근성 보강

## GitHub Pages 배포 절차

1. GitHub에 공개 저장소를 하나 생성한다.
2. 현재 프로젝트 폴더의 모든 파일을 저장소에 업로드한다.
3. 저장소 `Settings > Pages`에서 Source를 `main` 브랜치, `/ (root)`로 설정한다.
4. 생성되는 URL(예: `https://{계정명}.github.io/{저장소명}/`)을 과제 제출 창에 입력한다.
